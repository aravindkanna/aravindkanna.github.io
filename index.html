Major Projects

Here are some of my major projects.

Transitive Closure of Directed Graph

The project aims at computing the Transitive Closure of a Directed Graph in parallel. This is the project which introduced me to the parallel programming and OPENMP. Still in progress. Currently following a famous paper by J.D.Ullman. Working under the guidance of Prof. Kishore Kottapally from CSTAR of IIIT-H.

Face Detection

This is a course project done with the aim of detecting faces. As part of this project, I learned several ways of training a classifier and evaluating it by various methods of Cross Validation. Done following Viola Jones Algorithm.

Compute Eigen Values

This is the second project which I dealt in parallel. The aim of the project is to compute eigen values in parallel and I have used OpenMP for this with C++. Complete analysis was also added here.
